// Coleção de atalhos para engenheiros de rede e automação

Atalhos que uso no dia a dia para Python, Ansible, configuração de dispositivos e edição de código. Organizados por categoria para facilitar a consulta.

▸ Edição Básica
AtalhoO que faz
CTRL + XRecortar linha ou seleção
CTRL + CCopiar
CTRL + VColar
CTRL + DDuplicar linha (cursor solto) ou palavra (selecionada)
CTRL + ZDesfazer
CTRL + SHIFT + ZRefazer (retornar o que foi desfeito)
SHIFT + DELETEApagar linha inteira (sem seleção)
▸ Busca, Seleção Múltipla e Cursores
AtalhoO que faz
CTRL + FAbrir busca no arquivo atual
ALT + F3 (ou FN+ALT+F3)Seleciona todas as ocorrências da palavra sob o cursor
ALT + ENTERCria cursores em todas as ocorrências encontradas na busca
CTRL + Click EsquerdoCria cursores adicionais em pontos específicos (ou abre arquivos na Side Bar)
CTRL + ALT + Setas ↑/↓Cria cursor na linha acima/abaixo
Scroll Lock (botão do meio)Seleção vertical em bloco (modo coluna)
▸ Seleção e Movimentação Avançada
AtalhoO que faz
SHIFT + Setas ←/→Selecionar caractere a caractere
CTRL + SHIFT + Setas ←/→Selecionar palavra inteira (pula palavras)
CTRL + SHIFT + Setas ↑/↓Mover linha (ou bloco) para cima/baixo
▸ Side Bar e Grupos
AtalhoO que faz
CTRL + K + BAlternar visibilidade da Side Bar
SHIFT + ALT + 1Grupo único (voltar para única coluna)
SHIFT + ALT + 2Dividir em 2 grupos laterais (colunas)
SHIFT + ALT + 3Dividir em 3 grupos laterais
SHIFT + ALT + 4Dividir em 4 grupos laterais
▸ Formatação e Zoom
AtalhoO que faz
CTRL + K + UTransformar seleção em MAIÚSCULAS (Uppercase)
CTRL + K + LTransformar seleção em minúsculas (Lowercase)
CTRL + Scroll (frente/trás)Zoom in / Zoom out (aumenta/diminui fonte)
▸ Dicas de Produtividade
DicaDescrição
TABAuto-completar palavras já existentes no texto (usar setas para escolher)
Espaço vs. TabEspaço = caractere simples; Tab = avanço de célula (útil para dados tabulares)
CTRL + K + B + ClickCom Side Bar aberta, clique em arquivos com CTRL para abrir em nova guia
