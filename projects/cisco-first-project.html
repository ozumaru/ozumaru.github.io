PROJETO 01

Primeiro Projeto Prático

Este mini-projeto demonstra a automação de três tarefas essenciais em redes Cisco, com foco em refatoração de código para criar soluções limpas, reutilizáveis e fáceis de manter.

Desenvolvido durante a iniciativa Women Rock IT (WRIT) da Cisco, o projeto ensina a passar de scripts "bagunçados" para código profissional usando funções e instâncias (classes).

🚀 As 3 Tarefas do Projeto

// BACKUP
Rotina de Backup

Automatiza o backup de configuração de dispositivos Cisco de forma organizada e reutilizável.

// INTERFACE
Relatório de Interfaces

Gera relatório detalhado de status de interfaces (GigabitEthernet, Loopback, VLAN) com saída em JSON.

// VLAN
Criação de VLANs em Massa

Aplica VLANs automaticamente e coleta o estado do switch antes e depois da configuração.

🔄 Processo de Refatoração

  • → Criação de funções reutilizáveis
  • → Uso de instâncias (classes) para agrupar operações comuns
  • → Tratamento de saída de comandos Cisco para JSON estruturado
  • → Coleta de estado antes/depois das alterações
  • → Redução de código duplicado e melhoria na legibilidade
