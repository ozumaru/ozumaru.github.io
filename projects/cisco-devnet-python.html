O PROJETO

O que você vai encontrar aqui

Este repositório é um guia prático completo de automação de redes Cisco usando Python. Foi feito pensando em Network Engineers que querem sair do “clicar no CLI” e começar a automatizar tarefas reais.

Você vai aprender desde o básico (criar arquivos, usar data/hora, conexão SSH) até scripts mais avançados como criação de VLANs em massa, Loopbacks, links P2P e coleta de informações CDP exportadas em CSV.

📁 Estrutura do repositório

  • Documents/ → Passo a passo em .md (do básico ao avançado)
  • Projetos/ → Mini-projetos práticos prontos para usar
  • Netmiko + Python puro

Loopbacks e P2P automáticos
