PROJETO INTERNO

Reconciliação de Inventário NetBox

Projeto interno de automação avançada desenvolvido para garantir consistência total do inventário de rede no NetBox. Ele compara o estado desejado (CSV) com o estado atual no NetBox e realiza todas as criações e atualizações necessárias de forma hierárquica e idempotente.

✅ O que o sistema faz automaticamente

Manufacturers

Cria fabricantes se não existirem

Device Roles

Cria papéis (Switch, Router, etc.)

Device Types (Models)

Cria modelos de equipamentos

Sites & Locations

Gerencia sites e localidades

Devices + MGMT Interface

Cadastra dispositivos e interface de gerenciamento

IPv4 Management

Configura IP de gerenciamento e define como Primary

🏗️ Arquitetura em Camadas

Camadas principais do projeto:

  • Conexão API — Requisições REST seguras e padronizadas
  • Coleta de Dados — Leitura do CSV + consulta completa no NetBox
  • Validação Hierárquica — Verifica existência antes de criar (idempotente)
  • Orquestração — Fluxo controlado em `main_netbox.py`

STATUS: PROJETO INTERNO ATIVO

Este projeto demonstra minha capacidade de construir soluções de automação robustas, com arquitetura modular, validações fortes e integração profunda com NetBox.
